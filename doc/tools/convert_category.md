# convert_category.py - 分类转换工具

## 功能

转换指定分类下的所有页面，包括内容和页面名称。

## 用法

```bash
python src/convert_category.py [category_name]
```

### 参数

- `category_name`: 分类名称（默认：音樂）

## 示例

```bash
# 转换音乐分类
python src/convert_category.py "音樂"

# 转换其他分类
python src/convert_category.py "角色"
```

## 工作流程

1. 获取分类下的所有页面
2. 遍历每个页面：
   - 转换页面内容
   - 如果页面名称是繁体，移动到简体名称

## 特性

- 同时处理内容和页面名称
- 移动时不创建重定向
- 自动跳过图片字段

## 注意事项

- 会移动页面（修改页面名称）
- 移动后原页面名不再可用
- 可能会影响其他页面的链接
