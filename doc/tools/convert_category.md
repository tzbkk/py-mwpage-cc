# convert_category.py - 分类转换工具

## 功能

转换指定分类下的所有页面内容为简体中文。

## 用法

```bash
python src/convert_category.py <category_name> [选项]
```

### 参数

- `category_name`: 分类名称（不含 Category: 前缀）
- `--list`: 只列出分类下的页面
- `--dry-run`: 预览模式，不实际保存
- `--limit N`: 限制转换的页面数量
- `--no-test-first`: 跳过第一个页面的测试

## 示例

```bash
# 列出分类下的页面
python src/convert_category.py "音乐" --list

# 转换音乐分类
python src/convert_category.py "音乐"

# 预览转换
python src/convert_category.py "音乐" --dry-run

# 限制转换数量（用于测试）
python src/convert_category.py "音乐" --limit 5

# 跳过第一个页面测试
python src/convert_category.py "音乐" --no-test-first
```

## 工作流程

### 列出模式 (--list)
1. 获取分类下的所有页面
2. 显示页面列表

### 转换模式
1. 获取分类下的所有页面
2. 如果需要，先测试第一个页面
3. 批量转换页面内容
4. 显示统计信息

## 特性

- 列出模式查看分类下的页面
- 测试单个页面后再批量转换
- 限制数量用于测试
- 自动文件名保护验证
- 预览模式查看将要执行的修改
- 统计信息显示成功/失败/跳过数量
- 速率限制自动处理

## 推荐工作流程

1. **列出页面** - 查看分类下的所有页面
2. **限制测试** - 使用 --limit 5 测试前 5 个页面
3. **检查结果** - 到 Wiki 上确认转换正确
4. **批量转换** - 移除 --limit 进行完整转换

## 与其他分类工具的区别

| 工具 | 功能 | 处理对象 | 是否移动 |
|------|------|---------|---------|
| convert_category.py | 转换分类下的页面 | 分类下的页面 | ❌ |
| convert_cat_page.py | 批量转换分类页面 | 分类页面本身 | ❌ |
| convert_cat_ending.py | 批量转换并移动 | 分类页面本身 | ✅ |

## 注意事项

- 只转换页面内容，不移动页面
- 不会修改分类页面本身
- 建议先使用 --limit 5 测试
- 建议先使用 --dry-run 预览
- 建议先在小范围测试
- **脚本只修改 Wiki 页面内容，不会修改任何脚本文件**
