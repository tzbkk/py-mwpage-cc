# convert_seasons.py - 季度页面批量转换工具

## 功能

转换季度主页面及其所有子页面的内容和名称（如果需要）。

## 用法

```bash
python src/convert_seasons.py [page1] [page2] ...
python src/convert_seasons.py --from-file pages.txt
```

### 参数

- `page1, page2, ...`: 页面名称列表
- `--from-file FILE`: 从文件读取页面列表
- `--dry-run`: 预览模式，不实际保存和移动
- `--no-move`: 只转换内容，不移动页面
- `--no-subpages`: 不处理子页面

## 示例

```bash
# 转换指定页面
python src/convert_seasons.py "动画第一季" "动画第二季"

# 转换其他系列
python src/convert_seasons.py "漫画第一卷" "漫画第二卷"

# 从文件读取
python src/convert_seasons.py --from-file pages.txt

# 预览模式
python src/convert_seasons.py "动画第一季" --dry-run

# 只转换内容不移动
python src/convert_seasons.py "动画第一季" --no-move

# 不处理子页面
python src/convert_seasons.py "动画第一季" --no-subpages
```

### 文件格式

pages.txt 文件格式（以 # 开头的行会被忽略）：
```
动画第一季
动画第二季
# 这是注释
动画第三季
```

## 工作流程

1. **转换主页面**
   - 遍历指定的每个主页面
   - 转换内容为简体中文
   - 如果需要，移动主页面到简体名称

2. **转换子页面**
   - 查找每个主页面的子页面（如 "动画第一季/第1集"）
   - 转换子页面内容
   - 如果需要，移动子页面到简体名称

## 子页面识别

子页面是指使用 `/` 分隔的页面，例如：
- `动画第一季/第1集`
- `动画第一季/第2集`
- `动画第二季/第1集`

## 特性

- 支持批量处理多个主页面
- 从命令行参数或文件读取页面列表
- 自动查找并处理所有子页面
- 同时转换内容和移动页面
- 预览模式 (--dry-run) 查看将要执行的操作
- 可选不移动页面 (--no-move)
- 可选不处理子页面 (--no-subpages)
- 统计信息显示成功/失败/跳过数量
- 速率限制自动处理

## 使用场景

- 批量转换季度/系列页面
- 需要同时处理主页面和子页面
- 统一系列的批量处理

## 注意事项

- 会移动子页面（如果名称需要转换）
- 移动时不创建重定向
- 适合处理有规律命名的系列页面
- 建议先使用 --dry-run 预览
- 建议先在小范围测试
- **脚本只修改 Wiki 页面内容，不会修改任何脚本文件**
