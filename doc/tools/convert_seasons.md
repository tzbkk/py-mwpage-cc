# convert_seasons.py - 季度页面转换工具

## 功能

转换季度主页面及其所有子页面。

## 用法

```bash
python tools/convert_seasons.py [page1] [page2] ...
```

### 参数

- `page1, page2, ...`: 页面名称列表（默认：动画第一季, 动画第二季, 动画第三季）

## 示例

```bash
# 转换默认的三个季度页面
python tools/convert_seasons.py

# 转换指定页面
python tools/convert_seasons.py "动画第一季" "动画第二季"

# 转换其他系列
python tools/convert_seasons.py "漫画第一卷" "漫画第二卷"
```

## 工作流程

1. **转换主页面**
   - 遍历指定的每个主页面
   - 转换内容为简体中文

2. **转换子页面**
   - 查找每个主页面的子页面（如 "动画第一季/第1集"）
   - 转换子页面内容
   - 如果需要，移动子页面到简体名称

## 子页面识别

子页面是指使用 `/` 分隔的页面，例如：
- `动画第一季/第1集`
- `动画第一季/第2集`
- `动画第二季/第1集`

## 特性

- 自动查找并处理所有子页面
- 同时转换内容和移动页面
- 支持任意页面前缀

## 注意事项

- 会移动子页面（如果名称需要转换）
- 移动时不创建重定向
- 适合处理有规律命名的系列页面
