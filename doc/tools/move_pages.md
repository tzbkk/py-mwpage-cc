# move_pages.py - 页面移动工具

## 功能

批量移动/重命名页面到简体中文名称。

## 用法

```bash
python tools/move_pages.py [page1] [page2] ...
```

### 参数

- `page1, page2, ...`: 页面名称列表（可选）

## 示例

```bash
# 使用默认页面列表
python tools/move_pages.py

# 指定页面
python tools/move_pages.py "雖然不會說出口。" "愛歌"

# 批量处理
python tools/move_pages.py "頁面1" "頁面2" "頁面3"
```

## 工作流程

对于每个页面：

1. 将页面名转换为简体
2. 如果名称有变化：
   - 检查页面是否存在
   - 移动页面到新名称
   - 不创建重定向
3. 如果名称无变化，跳过

## 输出示例

```
移动: 雖然不會說出口。 → 虽然不会说出口。
  完成

移动: 愛歌 → 爱歌
  完成

无需移动: STARS

移动: 小小戀歌 → 小小恋歌
  页面不存在（可能已移动）

...

全部完成!
```

## 特性

- 只移动需要改名的页面
- 自动跳过已经是简体的页面
- 不创建重定向（避免冗余）

## 使用场景

- 批量重命名页面
- 繁体页面名转简体
- 统一页面命名规范

## 注意事项

- 移动后原页面名不再可用
- 可能影响其他页面的链接
- 移动操作不可逆
- 建议先在小范围测试
